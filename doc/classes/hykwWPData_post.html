iget_content()

iget_content(boolean  $isApplyFilter_content = TRUE) : string

iget_content 投稿の本文を返す

自動整形機能を無効にしないと、apply_filters()のタイミングで 勝手に整形(<br>や<p>が入る）されるので注意

# 自動整形機能を無効化
remove_filter( 'the_content', 'wpautop' );
remove_filter( 'the_excerpt', 'wpautop' );

Parameters

boolean $isApplyFilter_content

TRUE: apply_filters('the_content')を通す, FALSE: 通さない

Returns

string —

投稿の本文、投稿以外はFALSE
