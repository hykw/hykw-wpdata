iget_permalink()

iget_permalink(boolean  $isStripDomain = TRUE) : string

iget_permalink 選択中の固定ページのURLを返す（permalinkなので、アクセスURLに?code=などが付いていても、リターン値には含まれない）

実行例）

self::iget_permalink(TRUE)     # /page1
self::iget_permalink(FALSE)    # http://example.jp/page1

Parameters

boolean $isStripDomain

TRUE:ドメイン部分は返さない, FALSE:ドメイン部分も含む

Returns

string —

permalink(取得できない時は "")、固定ページじゃない時はFALSE
